Assignment 3
This program sorts an array using QuickSort, BubbleSort, and Shell sort

Build
make all

Running
./sorting
-a for all tests
-b for bubble test
-s for shell test
-q for Quicksort with Stack test
-Q for Quicksort with Queue test
-r to change seed
-n to set array size
-p to change elements to print out

Formatting
make format

Cleaning
make clean
