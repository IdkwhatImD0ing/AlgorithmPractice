# Error Correction
This program encodes and decodes files using hamming codes and corrects any errors that appear.

## Build
make all

## Running
./encode
-h for help page
-i to specify the infile
-o to specify the outfile

./decode
-h for help page
-v to specify if verbose printing is on or not
-i to specify the infile
-o to specify the outfile

./error
-h for help page
-e error rate
-s seed

./entropy
-h for help page

## Formatting
make format

## Cleaning
make clean
