Assignment 6 Huffman Coding
This program compresses and decompresses files.
Build
make all

Running
./encode
-h for help page
-v to print statistics
-i to specify the infile
-o to specify the outfile

./decode
-h for help page
-v to print statistics
-i to specify the infile
-o to specify the outfile


./entropy
-h for help page

Formatting
make format

Cleaning
make clean
